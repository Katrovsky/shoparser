# How to
---

## 1 Clone this repo
```bash
git clone https://github.com/Katrovsky/shoparser
```
## 2 Change directory
```bash
cd shoparser/
```
## 3 Make Python environment & install requirements
```bash
python -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt
```
## 4 Run it!
```bash
python Shoparser.py
```
